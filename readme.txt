Display the latest tweet from any Twitter account.

== Description ==

This plugin will retrieve the latest tweet from a <a href="http://www.twitter.com">Twitter</a> account and display it anywhere you would like in your template.

== Installation ==

1. Download and extract the plugin's zip file.
2. Upload the extracted `sg-tweet` folder to your `wp-content/plugins` directory.
3. Activate the plugin and enter `<?php sgLatestTweet('YOURTWITTERUSERNAME'); ?>` in your template where you would like to display your latest Tweet
